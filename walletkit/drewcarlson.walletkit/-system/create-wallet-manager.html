final Boolean createWalletManager(Network network, WalletManagerMode mode, AddressScheme addressScheme, Set<Currency> currencies)
Content copied to clipboard

Create a wallet manager for network using `mode.

Note: There are two preconditions - "network" must support "mode" and "addressScheme". Thus a fatal error arises if, for example, the network is BTC and the scheme is ETH.

Return

true on success; false on failure.

Parameters

network

the wallet manager's network

mode

the wallet manager mode to use

addressScheme

the address scheme to use

currencies

the currencies to register. A wallet will be created for each one. It is safe to pass currencies not in "network" as they will be filtered (but bad form to do so). The "primaryWallet", for the network's currency, is always created; if the primaryWallet's currency is in currencies then it is effectively ignored.
